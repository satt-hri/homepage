## 技术细节

### 使用的库
- **AOS**: 滚动触发动画 (https://github.com/michalsnik/aos)
- **自定义CSS**: 纯CSS动画和过渡效果

### 动画配置
```javascript
AOS.init({
    duration: 800,        // 动画持续时间
    easing: 'ease-out-cubic',  // 缓动函数
    once: true,           // 只触发一次
    offset: 100,          // 触发偏移量
    delay: 0,             // 默认延迟
    mirror: false,        // 不反向动画
    anchorPlacement: 'top-bottom'  // 触发位置
});
```

### 性能优化
- 使用 `transform` 和 `opacity` 实现硬件加速
- 动画只触发一次，避免重复计算
- 响应式设计，移动端禁用部分动画
